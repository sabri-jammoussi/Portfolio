Compétences informatiques

HTML5

HTML5 est la dernière version du langage de balisage pour la création de pages web, offrant de nouvelles balises sémantiques, des fonctionnalités multimédias intégrées, et une meilleure prise en charge des applications interactives sans nécessiter de plugins tiers.

Vue JS

Conçu pour être simple et modulaire, Vue.js offre une syntaxe intuitive et des outils puissants comme le data-binding réactif, le système de composants et la gestion des états pour développer des applications web modernes.

CSS3

CSS3 est la dernière évolution du langage de styles, introduisant de nouvelles fonctionnalités comme les transitions, les animations, les transformations 3D, et les media queries pour concevoir des interfaces web modernes et responsives.

ASP.NET Core 8

ASP.NET Core 8 est une plateforme open-source et multiplateforme pour développer des applications web modernes, offrant des performances élevées, une architecture modulaire, et un support intégré pour les API REST, les applications en temps réel, et le cloud.

GitHub

GitHub est une plateforme de collaboration et de gestion de versions basée sur Git, permettant aux développeurs de stocker, partager, et suivre les modifications de leurs projets tout en favorisant le travail d'équipe.

SQL Server Management Studio (SSMS)

SQL Server Management Studio (SSMS) est un outil intégré de gestion et d'administration des bases de données SQL Server, permettant aux développeurs et administrateurs de créer, configurer, gérer, et interroger des bases de données de manière intuitive.

Node Js

Node.js est une plateforme open-source, basée sur le moteur JavaScript V8 de Chrome, permettant d’exécuter du code JavaScript côté serveur pour créer des applications web rapides, scalables et non-bloquantes.

Flutter

Flutter est un framework open-source développé par Google, permettant de créer des applications multiplateformes performantes avec une seule base de code, pour Android, iOS, web et desktop.

Docker Compose

Docker Compose est un outil permettant de définir et de gérer des applications multi-conteneurs Docker, à l'aide d'un fichier de configuration YAML (docker-compose.yml) pour orchestrer le déploiement, la gestion des services et la mise en réseau entre conteneurs.
